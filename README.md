Medical Image Processing and Registration (APS)

This project focuses on medical image processing and image registration techniques used in angiographic and DSA workflows. The system is designed to analyze motion, align image frames, and improve visualization of vascular structures.

Overview
This project implements a pipeline for preprocessing, motion analysis, and image registration. It is useful for understanding how medical imaging data can be enhanced and aligned for better clinical interpretation.

Key Features
- Image preprocessing and normalization
- Motion analysis for sequential medical images
- Frame alignment using registration techniques
- Support for DSA style processing workflows
- Visualization of intermediate and final outputs

Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

Project Structure
- APS.py and related scripts handle processing pipeline
- DAS_motion.py handles motion analysis
- test_registration.py used for validation and testing
- Jupyter notebooks for experimentation and visualization

Note
Datasets and output files are not included due to size constraints. The repository contains only the core implementation and logic.

Applications
- Medical imaging analysis
- DSA (Digital Subtraction Angiography)
- Motion correction in image sequences
- Research and development in healthcare imaging

Future Improvements
- Real-time processing support
- Integration with 3D reconstruction pipelines
- Optimization for clinical workflows
